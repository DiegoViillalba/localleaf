Funcionalidades

Todo lo que necesitas.
Nada que no necesites.

Diseñado para investigadores, estudiantes y escritores que quieren foco, no fricción.

Privacidad total

Tu código LaTeX nunca sale de tu equipo. Sin cuentas, sin telemetría, sin sincronización forzada en la nube.

Tectonic integrado

No necesitas instalar TeX Live ni MiKTeX. Tectonic viene empaquetado y descarga solo los paquetes que usa tu documento.

IA compatible con cualquier proveedor

Conecta OpenAI, Gemini, LM Studio u Ollama. El asistente conoce tu documento y puede corregir errores de compilación automáticamente.

Proyectos multi-archivo

Árbol de archivos real con soporte para \input e \include. Define el archivo raíz y edita cualquier subarchivo sin perder el contexto.

Gestión de bibliografía

Importa referencias por DOI, arXiv o ISBN con un clic. Edita tus entradas BibTeX con una interfaz amigable y validación en tiempo real.

Editor con superpoderes

CodeMirror 6 con resaltado LaTeX, autocompletado de comandos, linter de entornos \begin/\end y navegación por el esquema del documento.

Descargas

Elige tu plataforma

Instaladores nativos para macOS, Windows y Linux. Sin instalación de dependencias adicionales.

Alfa privada. Esta versión es para testers invitados. Puede contener bugs y funciones incompletas. Tu feedback es clave para mejorar el producto. Reporta problemas en GitHub Issues.
